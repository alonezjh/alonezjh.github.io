Android Native + React Native 混合开发（一）：RN嵌入Android原生项目

字数 1,489 | Android+RN | | 发表于 2016-12-16 | 浏览数

前言

由于公司研发团队想要技术栈统一，因此项目中要求接入React Native开发。虽然之前听说过React Native的强大，但奈何此前没有任何的RN实战开发经验，无奈之下只能赶鸭子上架，一边继续Android原生开发，一边学习着RN。不得不说的是，RN在Windows下对Android应用开发着实不太友好，主要体现在版本的混乱以及各种莫名其妙的Bug。幸运的是国内也涌现出了大量的RN学习教程，遇到的问题基本上也能一一解决。虽然RN学习教程挺多，但是Android原生和RN的混合开发教程却是较少。因此，笔者准备在此记录并分享一些Android原生与RN混合开发的经验，基于Window下开发环境。该Android Native + React Native 混合开发系列教程乃笔者个人学习与开发经验，如有不足之处，敬请谅解。
